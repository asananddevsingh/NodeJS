
/*
  Commands to install npm packages.

  npm init                              : It is used to create/configure "package.json" file for your application.
  npm install                           : It will install all the "dependencies" specified at "package.json".
  npm install --dev                     : It will install all the "devDependencies" specified at "package.json".
  npm install <package-name> --save     : It will install the package and add it in "dependencies" object of the "package.json" file.
  npm install <package-name> --save-dev : It will install the package and add it in "devDependencies" object of the "package.json" file.

*/
